# Case-9-Websocket-Canvas

För att använda applikationen en användare behöver 
1. clona ner repot
2. köra npm install
#npm install ws?
3. npm run dev 
4. Öppna localhost:8081 i två olika webbläsaren 
5. ansluta till spelet och chatten genom att skriva sitt namn i formuläret
6. spela med andra spelare genom att röras sig på höger, vänster, upp och ner. 
7. chatta med andra användare genom chatten
